ArchLinux安装教程

ArchLinux是一个Linux发行版，使用Pacman包管理器，如果你要安装这个发行版并将主力系统迁移至ArchLinux的话，请确保你对Linux有足够的了解。

本教程仅针对实体机上的单系统安装，仅针对对Linux有一定了解的用户。

桌面环境使用KDE，使用XFS，使用Intel CPU，使用Nvidia较新的独显，不开启Swap分区。

首先，在安装前，我们要准备一个1GB以上的U盘。然后去国内镜像站，比如USTC Mirrors下载最新镜像，使用Rufus将ISO镜像烧录到U盘中。

Boot into USB

将U盘插入电脑，然后进入Bios切换引导至U盘。

系统时钟

进入系统后，首先要更新系统时钟。

timedatectl set-ntp true

更换国内镜像源

然后更换国内软件仓库镜像源加快下载速度。

vim /etc/pacman.d/mirrorlist

在最开头添加

Server = https://mirrors.ustc.edu.cn/archlinux/$repo/os/$arch

分区和格式化

首先，最基本的分区有2个，一个主分区，一个Boot分区。

使用fdisk -l查看你要安装的硬盘，在这里假设是**/dev/nvme0n1**

我们使用gdisk进行分区，执行gdisk /dev/nvme0n1，后输入o, 意味着create a new empty GUID partition table (GPT)，回车。

然后，打n，新建分区，回车2次，在Last sector （截止扇区）键入：+512MB，UEFI启动系统会在这个新建的EFI分区里储存引导文件。

接下来Hex code or GUID (L to show codes, Enter = AF00): 这里打 EF00，因为我们要efi分区。

然后再n一次，一路回车可以直接把剩下所有区块吃完。

然后打w，写入之前的操作。

此时，Boot分区分区应该是**/dev/nvme0n1p1，而主分区则是/dev/nvme0n1p2**。

我们使用mkfs.fat格式化Boot分区，使用mkfs.xfs格式化主分区。

mkfs.fat -F32 /dev/nvme0n1p1
mkfs.xfs -f /dev/nvme0n1p2

接下来我们将主分区挂载到/mnt上，并创建/mnt/boot，然后将Boot分区挂载到/mnt/boot上。

mount /dev/nvme0n1p2 /mnt
mkdir /mnt/boot
mount /dev/nvme0n1p1 /mnt/boot

pacstrap

输入以下命令安装。

pacstrap /mnt base base-devel linux linux-firmware dhcpcd vim intel-ucode nvidia

Fstab

genfstab -U /mnt >> /mnt/etc/fstab

进入系统

输入以下命令进入安装的Arch Linux。

arch-chroot /mnt

引导

在这里，我不使用Grub，而使用systemd的bootctl。

bootctl install
cd /boot/
cd loader/
vim loader.conf

将loader.conf的内容更改为：

default arch
timeout 0

接下来

cd entries/
vim arch.conf

将arch.conf的内容更改为：

title   Arch Linux
linux   /vmlinuz-linux
initrd  /intel-ucode.img
initrd  /initramfs-linux.img
options root=/dev/nvme0n1p2 rw

至此，引导便完成了。

设置主机名、Hosts与时区

首先在/etc/hostname设置主机名：

vim /etc/hostname

将hostname的内容更改为主机名。这里假设输入的主机名是arch

接下来在/etc/hosts设置与其匹配的条目：

vim /etc/hosts

将hosts的内容更改为:

127.0.0.1   localhost
::1         localhost
127.0.1.1   arch.localdomain	arch

随后设置时区，在/etc/localtime下用/usr中合适的时区创建符号链接。

ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

使用hwclock –systohc将系统时间同步到硬件时间。

设置 Locale

Locale决定了软件使用的语言、书写习惯和字符集。

编辑/etc/locale.gen，去掉en_US.UTF-8 UTF-8以及zh_CN.UTF-8 UTF-8行前的注释符号。

vim /etc/locale.gen

然后使用locale-gen生成locale。

向 /etc/locale.conf 输入内容：

echo 'LANG=en_US.UTF-8'  > /etc/locale.conf

为 root 用户设置密码

passwd

更改wheel

输入命令visudo，找到如下这样的一行，把前面的注释符号去掉：

#%wheel ALL=(ALL) ALL

创建用户

我们这里假设你创建的用户名叫xnn。

useradd -m -G wheel xnn

然后使用passwd xnn更改密码。

重启

输入exit退出，并使用reboot重启。然后把U盘拔掉。

重启后使用新创建的用户登陆。

安装桌面环境和功能包

sudo pacman -S xorg-server plasma-meta konsole dolphin ark okular gwenview kate spectacle wget 
sudo pacman -S noto-fonts noto-fonts-cjk noto-fonts-extra noto-fonts-emoji ttf-dejavu ttf-liberation
sudo pacman -S fcitx5-im fcitx5-chinese-addons fcitx5-pinyin-zhwiki
sudo pacman -S networkmanager

编辑/etc/environment，添加：

GTK_IM_MODULE=fcitx
QT_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx
INPUT_METHOD=fcitx
SDL_IM_MODULE=fcitx
GLFW_IM_MODULE=ibus

添加ArchLinuxCN

编辑/etc/pacman.conf，在末尾添加：

[archlinuxcn]
Server = mirrors.ustc.edu.cn/archlinuxcn/$arch

输入sudo pacman -Syyu更新，然后输入sudo pacman -S archlinuxcn-keyring。

安装yay并安装Chrome浏览器

sudo pacman -S yay
yay -S google-chrome

Systemctl enable

systemctl enable sddm
systemctl enable dhcpcd
systemctl enable NetworkManager

进入图形界面

重启电脑后就可以进入图形界面了。

完毕.
